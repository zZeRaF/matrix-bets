🔄 SYNCHRO MULTI-DEVICE

🔄 Configurer la synchro

Cette PWA synchronise ta bankroll et tes paris entre tous tes appareils via un Gist GitHub privé. Le Gist n'est visible que par toi (compte GitHub).

Étape 1 — Créer un Personal Access Token

  1. Ouvre github.com/settings/tokens/new
  2. Note (n'importe quel nom) : matrix-bets-sync
  3. Expiration : No expiration (ou la durée qui te convient)
  4. Coche uniquement la case gist
  5. Tout en bas, clique Generate token
  6. Copie le token affiché (commence par ghp_)

Étape 2 — Coller le token

Le token est stocké sur ce device uniquement. Pour configurer un autre device, tu colleras le même token là-bas.
